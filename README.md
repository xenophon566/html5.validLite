ValidLite
=========

#### Type
Frontend Plugin

#### Describe
A Frontend validation plugin.

#### Feature
1.	Easy to use.
2.	Using data-* to validate your fields.
3.	Customize validation pattern for each field.
4.	Customize validate messages and styles.

#### Base on
HTML5, CSS3, Javascript, JQuery
